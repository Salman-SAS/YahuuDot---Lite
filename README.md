⚡ The Problem

Every day, thousands of users fall victim to:

Phishing emails pretending to be banks
Fake delivery notifications
Social engineering scams
Impersonation messages on WhatsApp & email

Most users cannot detect these threats in time.



🧠 The Solution

YahuuDot Lite is an AI fraud detection agent that analyzes suspicious messages in real time and tells you:

Is this a scam or safe?
How dangerous is it?
What tricks are being used?
What should you do next?
🛡️ How It Works
Paste any suspicious message, email, or link
AI analyzes content using Google Gemini
System detects fraud patterns:
Phishing
Impersonation
Urgency manipulation
Credential theft attempts


Returns structured intelligence report:
Threat Summary
Risk Score (LOW → CRITICAL)
Indicators
Recommended Actions
📊 Risk Intelligence Engine
🟢 LOW → Safe or minimal risk
🟡 MEDIUM → Suspicious patterns detected
🟠 HIGH → Likely scam attempt
🔴 CRITICAL → Immediate threat detected
